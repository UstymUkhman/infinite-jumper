# [Infinite Jumper](https://ustymukhman.github.io/infinite-jumper/public) #

## Download ##

`git clone https://github.com/UstymUkhman/infinite-jumper.git`

`cd infinite-jumper`

## Develop ##

`yarn`

`yarn dev`

## Build ##

`yarn build`
